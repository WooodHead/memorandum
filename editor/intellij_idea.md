## Shortcut key
* [Mastering IntelliJ IDEA keyboard shortcuts - Help | IntelliJ IDEA](https://www.jetbrains.com/help/idea/mastering-intellij-idea-keyboard-shortcuts.html)

* `Cmd+,`
    * Preference
* `Cmd+Shift+A`
    * commanの検索
* `Cmd+N`
    * find class
* `Cmd+Shift+N`
    * find file
* `option+Shift+N`
* `option+F1`
    * 現在のfileを指定のviewで開く
* `Cmd+F4`
    * close tab
* `Ctlr+arrow`
