# riv

## Install

```vim
NeoBundle `Rykka/riv.vim`
```

## Settings

```vim
let g:riv_fold_level = 1
"let g:riv_web_browser = '/path/to/broweser'
let g:riv_temp_path = 0
let g:riv_html_code_hl_style = "friendly"
```

## Usage

* htmlでのpreview

```vim
:Riv2HtmlAndBrowse
```
