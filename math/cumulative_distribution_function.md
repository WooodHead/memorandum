## Cumulative distribution function
* $F: \mathbb{R} \rightarrow \mathbb{R}$,

### Definition

$F$ is said to be **non increasing** if

$$
    \forall x < y,
    \
    F(x) \ge F(y)
    .
$$

$F$ is said to be **non decreasing** if

$$
    \forall x < y,
    \
    F(x) \le F(y)
    .
$$
