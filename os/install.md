## install

```
install [OPTION]... SOURCE DEST
install [OPTION]... SOURCE... DIRECTORY 
install [OPTION]... -t DIRECTORY SOURCE
install [OPTION]... -d DIRECTORY
```

* `-d`
    * 引数を全てdirectory名として解釈
    * directoryを作成する

必要なdirectoryを全て作成

```
install -d directory
```
