## getent
get entries from administrative database.

* supported databases
    * ahosts
    * ahostsv4
    * ahostsv6
    * aliases
    * ethers
    * group
    * gshadow
    * hosts
    * initgroups
    * netgroup
    * networks
    * passwd
    * protocols
    * rpc
    * services
    * shadow


## CLI

```
getent group GROUPNAME
```
