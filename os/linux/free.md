## free

Show memory usage megabytes

```
free [option]
```

* `-b`
    * bytes
* `-m`
    * megabyte
* `-g`
    * gigabyte

```
/ # free -m
             total       used       free     shared    buffers     cached
Mem:         15044       9737       5307          1       1772       3396
-/+ buffers/cache:       4568      10476
Swap:            0          0          0
```
