## host
host command.

```
host [hostname] server
```

localのDNS serverでhostの問い合わせ

```
host hostname localhost
```

* `-a`
    * all
* `-d`
    * debug mode
* `-v`
    * same sa debug
* `-t query_type`
    * "a"
    * "ns"
    * "md"
    * "mf"
    * "cname"
    * "soa"
    * "mb"
    * "mg"
    * "mr"
    * "null"
    * "wks"
    * "ptr"
    * "hinfo"
    * "minfo"
    * "mx"
    * "uinfo"
    * "uid"
    * "gid"
    * "unspec
