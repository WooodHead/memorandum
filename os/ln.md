## ln

```
ln [OPTION]... [-T] TARGET LINK_NAME (1番目の形式) 
ln [OPTION]... TARGET (2番目の形式) 
ln [OPTION]... TARGET... DIRECTORY (3番目の形式) 
ln [OPTION]... -t DIRECTORY TARGET... (4番目の形式)  
```


* `-L, --logical`
* `-n, --no-dereference`
* `-s, --symbolic`
    * sybolic link
* `-T, --no-target-directory`
    * 常に LINK_NAME を通常ファイルとして扱う
* `-f, --force`
