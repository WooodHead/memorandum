## tfenv
rbenv for terrafomr.

## Install

```
$ brew install tfenv
```

## Usage

```
tfenv install 0.7.0
```

```
tfenv use min-required
```

```
tfenv uninstall 0.7.0
```

installed version

```
tfenv list
```

installable version

```
tfenv list-remote
```
