# ag

## Install

### mac

```shell
brew install ag
```

## usage
hoge を文字列として含むファイルをカレントディレクトリから再帰的に調べる。

```shell
ag hoge
```

| オプション | 意味                               |
|------------|------------------------------------|
| -a         | 無視リストも対象                   |
| -u         | 隠しファイルおよび無視リストも対象 |
| -g         | ファイル名で検索                   |
| -G         | 検索対象の指定 [ag -G *.css]       |
| -l         | 一致したファイル名のみ             |
| -L         | 一致していないファイル名のみ       |
| -v         | 不一致検索                         |
| -Q         | 正規表現を無視                     |
| -z         | 圧縮ファイルも対象                 |
