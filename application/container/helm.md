## Helm
The Kubernetes Package Manager.

* helm
    * client
* tiller
    * server
    * kubernetes clusterの中で動く
* Chart
    * Helm package
* Repository
    * chartの集まり
    * CPANなどと同じ
* Release
    * chartのinstanceでkubernetes clusterの中で動作する
