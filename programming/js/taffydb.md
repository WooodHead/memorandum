## taffydb
javascriptのdbで、brwoserとserver sideで使えるin memory DB.

## Usage

Create DB

```javascript
var product_db = TAFFY([
  { "item"  : 1,
    "name"  : "Blue Ray Player",
    "price" : 99.99
  },
  { "item"  : 2,
    "name"  : "3D TV",
    "price" : 1799.99
  }
]);
```
