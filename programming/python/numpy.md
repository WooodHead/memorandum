## numpy

## function
* `vstack(tup)`
    * 配列を行に連結
* `hstack(tup)`
    * 配列を列方向に連結
* `logical_xor`
    * 2つの配列の要素ごとの排他的論理和をとる
* `newaxis`
    * 配列に新しい次元を挿入したいときに利用。
    * `[:, np.newaxis]`で1次元配列が2次元配列になる

## operators
* 配列に対して、条件比較ができる
    * 要素ごとの条件比較に置きかわる
