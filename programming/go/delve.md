## Delve
Debugger

## Install

```
go get -u github.com/derekparker/delve/cmd/dlv
```

For OSX

```
go get -d github.com/derekparker/delve/cmd/dlv
cd ${GOPATH}/src/github.com/derekparker/delve
make install
```
